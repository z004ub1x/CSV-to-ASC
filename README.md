I created this script to help a customer convert large quantities of CSV files into a format directly readable into Simcenter Testlab.  This follows the header formatting specified in https://support.sw.siemens.com/en-US/product/215806642/knowledge-base/KB000039033_EN_US (requires Siemens Support Center account to access).  The script assumes the first row is the header row, containing only channel names.  Optionally, it can interpret the first column of data as absolute time stamps, and add absolute time to the ASC header info; this feature also interprets the sample rate by calculating the delta between the first several samples.  The absolute time stamp formats supported do not specify the date, so there is also a feature where by (depending on which format is detected) the user can specify the date the recording was made, and the absolute time/day stamp is automatically calculated and added to the ASC header.  Example CSV and ASC files are also attached for reference and practice with using the script.
